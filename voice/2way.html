This version uses a presentation-style layout: top navigation, a full visual stage, and a slim sticky control rail on the left. It keeps the live demo feeling closer to a premium showcase instead of a control panel.

Premium presentation layout The demo stage stays central while the commands stay visible but lightweight.

Why this layout works better live

The main stage no longer competes with a dense control panel. This is closer to how presentation tools treat controls: always available, but visually secondary to the content you are showing.
